# Data Structures Mini-Course

Quick refresher notes for the `data_structures` repo. Think of this as a 30‑minute tour you can skim before an interview or coding exercise.

## How to Use
1. Skim `cheat_sheet.md` for the high-level concepts and time-complexity tables.
2. Jump into the referenced implementation files under `src/` when you want to see code.
3. Pick one practice prompt per section (listed in the cheat sheet) and run it locally for spaced repetition.

## Sections at a Glance
| Section | What you get | Primary references |
| --- | --- | --- |
| 1. Linear Containers | Arrays vs linked lists trade-offs | `src/data_structures/linked_list.py`, interview array problems |
| 2. Linked Lists | Singly vs doubly nuances, in-place tricks | `src/data_structures/linked_list.py`, linked list interview files |
| 3. Stacks & Queues | LIFO/FIFO patterns, deque tips | `src/data_structures/stack_and_queue.py` |
| 4. Hash Tables & Sets | Collision handling, amortized costs | `src/data_structures/hash_table.py`, `src/interview/group_anagrams.py` |
| 5. Trees & BSTs | Traversals, balancing intuition | `src/data_structures/binary_search_tree.py`, BST interview set |
| 6. Heaps & Priority Queues | Min/Max heap ops, selection problems | `src/data_structures/min_heap.py`, `src/interview/kth_largest_number.py` |
| 7. Graphs | BFS/DFS mental models | `src/data_structures/graph.py`, graph-flavored interview prompts |
| 8. Algorithmic Patterns | Sliding window, divide & conquer, heap + hash combos | Various interview files |

Each section in `cheat_sheet.md` is intentionally terse—just enough to jog your memory before diving back into implementations.
